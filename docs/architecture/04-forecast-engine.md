# Forecast Engine

## Model

Deterministic projection based on:

- Average daily inflow
- Average daily outflow
- Trend rate
- Recurring transactions

## Confidence Bands

Upper = baseline + 1 std deviation  
Lower = baseline - 1 std deviation  

Confidence = 68%

## Why Not ML?

Deterministic forecast provides:
- Explainability
- Auditability
- Financial reliability
